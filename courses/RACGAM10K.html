Cursusdoelen
In deze module leer je wat een serious game is en in welke situaties een serious game uitkomst biedt. Je leert een innovatief concept te bedenken voor een specifiek leerdoel uit je eigen opleiding of werkpraktijk. Je gaat hands-on aan de slag in meerdere sprints om al snel je idee te visualiseren en te testen. Iedere sprint is een doelgerichte stap die ervoor zorgt dat je steeds dichter bij je einddoel komt. Uiteindelijk demonstreer je jouw serious game in een prototype.
Uiteraard onderbouw je waarom jouw serious game de innovatieve oplossing vormt voor het specifieke praktijkprobleem.
 
Omdat je gebruik maakt van bestaande tools is het niet nodig om te kunnen programmeren. 
 
Inhoud
Verantwoording voor de student:
Serious games zijn als moderne managementtechniek in te zetten om specifieke praktijkproblemen het hoofd te bieden. Het is een innovatieve oplossing voor een specifiek handelingsprobleem.
 
Doelen cursus voor de student :
Dublin1 kennis: student leert wat een serious game is, hoe je dmv een iteratief proces een prototype kunt ontwikkelen.
Dublin2 toepassen kennis: student leert hoe je een serious game kunt toepassen in werksituaties
Dublin4 communicatie: een serious game wordt meestal ontwikkeld in een multidisciplinair team. Welke rol vervult 1-ieder daarin en welke inbreng wordt verwacht.
 
Na afloop is de student in staat om de kenmerken van een serious game te benoemen. Tevens kan de student aangeven in welke praktijksituaties een serious game uitkomst kan bieden. De student kan na het volgen van deze module beschrijven wat een iteratief proces is en hoe dit proces helpt om een serious game te ontwikkelen. De student kan een innovatieve oplossing bedenken voor een specifiek handelingsprobleem. Na afloop is de student in staat om d.m.v. een prototype de werking van de game te demonstreren en het concept te onderbouwen.
 
 
Beginniveau voor de student[G3] :        er zijn geen specifieke eisen. Iedereen kan instromen.
 
 
Inhoud voor de student :
Les 1: introductie; wat is een serious game incl. voorbeelden
Les 2: theorie ontwerpen van een serious game / brainstormsessie hoe verzin je een concept
Les 3: theorie ontwerpen van een serious game / prototyping / eerste sprint schetsen op papier
Les 4: workshop ontwerpen van een serious game; presentaties van eerste ontwerp per groepje
Les 5: instructie tool voor bouwen serious game prototype / tweede sprint
Les 6: workshop testen van een prototype
Les 7: workshop bouwen van een serious game prototype / derde sprint
Les 8: eindpresentaties

Toetsvorm: individuele digitale kennistoets voor 30 punten & groepsopdracht voor 70 punten.
De opdracht luidt: ontwikkel een prototype van een serious game met behulp van tools en maak daarbij een schriftelijke onderbouwing van het concept waarin je laat zien dat je de theorie van serious gaming kunt toepassen.

 In de verantwoording wordt aangegeven waarom het keuzecursus is opgenomen in het hogeschoolbrede keuzeonderwijs. De volgende argumenten komen daarvoor in aanmerking:
a) De cursus sluit aan bij de instituutsprofilering: thema's en subthema's van onderzoek, kennis-centrum, innovation labs etc.;
b) De cursus is een inleiding van de corebusiness van een opleiding van het instituut;
c) De cursus ondersteunt de student bij het behalen van de studie (=ondersteunend onderwijs, learning skills, persoonlijke effectiviteit).
Daarnaast wordt de bruikbaarheid van het keuzecursus voor hbo-functies aangetoond door in de verantwoording de hbo-functies te noemen waarop de keuzecursus zich richt.door het instituut
De cursus heeft een helder (smart) omschreven leerdoel waarbij met behulp van de 5 Dublin descriptoren (kennis/inzicht, toepassen kennis/inzicht, oordeelsvorming, communicatie en leervaardigheden) specifiek wordt benoemd welke doelen worden nagestreefd. De doelen tonen op deze manier het hbo-niveau aan. Werk-, toetsvorm en literatuur sluiten aan op de gestelde doelen (zie bijlage "Dublin descriptoren HBO-Bachelor.docx").
 
 In beginsel staan keuzecursussen open voor alle studenten van de hogeschool. Uitzonderingen zijn mogelijk, bijvoorbeeld als keuzecursussen specifiek bedoeld zijn voor eerstejaars studenten of als er een bepaalde volgtijdelijkheid is (niveau keuzecursus x1 als voorwaarde voor keuzecursus x2).
 
 De cursus heeft een helder omschreven inhoud waarbij specifiek wordt benoemd welke onderwerpen aan de orde komen.
 De cursus geeft een duidelijke omschrijving van de werkvormen, die aansluit bij de doelen en toetsvorm.
 De cursus geeft een duidelijke omschrijving van de toetsvorm en hoe die aansluit bij de doelen, de werkvorm en de literatuur.
 
 De cursus geeft een specificatie van de te bestuderen literatuur (auteur, titel, hoofdstukken, aantal pag. etc.).
